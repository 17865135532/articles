# 科技爱好者周刊：第 75 期

现在，全世界的汽车公司，都开始制作电动汽车。美国“汽车城”底特律的[新闻媒体](https://www.detroitnews.com/story/business/autos/2019/09/05/shift-electric-vehicles-radically-change-auto-factories/2208961001/)非常担心，认为这会大量失业。

原因很简单。传统汽车是一个机械装置，需要大量零件，单单是动力传动系统就多达2,000个零件。但是，电动汽车是一个电子装置，核心是一个电动机和一个大电池，零件数量大大减少。变速器、散热器、燃油喷射器、油箱、排气系统，都不需要。

而且，电动机和电池都有专门厂商标准化生产。所以，未来的汽车公司，很可能与今天的电脑公司和手机公司差不多：核心部件都是采购来的，自己只负责外面的壳子和内部的软件系统，把所有东西组装在一起。

这样一来，汽车公司根本不需要那么多工人，尤其是跟机械装置相关的岗位，都会消失。

顺着这个思路设想，以后不管是什么东西，很可能都是电子装置，因为需要控制电路、联网和运算能力。比如，智能电饭煲、智能运动鞋、智能马桶的核心都是电子装置。

这意味着，未来产业发展的方向，主要就是三个产业：（1）能源和电池，为电子装置提供能量；（2）电子设备和通信设备，作为电子装置的硬件载体；（3）软件业，作为电子装置的控制和互动系统。因此，个人的发展路线，尤其是工程师，应该往这些方向靠。

## 文摘

1、[时空旅行者宴会](https://baike.baidu.com/item/%E6%97%B6%E9%97%B4%E6%97%85%E8%A1%8C%E8%80%85%E5%AE%B4%E4%BC%9A)

2009年6月28日，英国著名物理学家霍金举行了一次时间旅行者宴会。房间布置了气球、香槟和美食，挂著大大的标语，写着“欢迎时间旅行者”。

![](https://www.wangbase.com/blogimg/asset/201908/bg2019081104.jpg)

霍金在宴会举办前没有向任何人发出邀请。宴会结束后，他才发出请帖，邀请有“穿越”能力的人士赴宴。

![](https://www.wangbase.com/blogimg/asset/201908/bg2019081106.jpg)

请帖上写著：“诚挚邀请你参加时间旅行者的宴会。宴会由史蒂芬·霍金教授举办。”请帖不但写明宴会的举办地点为英国剑桥大学冈维尔与凯斯学院，还贴心地标明了经纬度。

霍金认为，如果有未来人看到这份请帖，并且能“穿越”回到过去，那么他在那次宴会上就会见到货真价实的“时间旅行者”。

![](https://www.wangbase.com/blogimg/asset/201908/bg2019081105.jpg)

但是，完全没有人赴会。霍金说：““我坐了很长时间，但没有人来。我用实验证明，时间旅行不可能。”

1905年，爱因斯坦在相对论中断定，当接近光速的运动时，时间会很慢或静止，也就是说，当人乘坐接近光速的飞船去旅行，在旅行的过程中时间就会变慢，因此，当他再回到地球的时候就可能已经过了一个世纪。对他来 说，只要花很少的时间就能进入未来世界。但是，爱因斯坦指出：光速无法超越，时间不可能倒流。

不过，霍金说：“从爱因斯坦的相对论来看，弯曲时空、回到过去的可能性的确存在。但是，让时空弯曲，可能引发能量摧毁（时空旅行所用的）飞船、甚至时空。”　

2、[马斯克的脑机接口](https://arstechnica.com/science/2019/08/elon-musks-neuralink-both-an-evolution-and-a-plan-for-radical-change/)

![](https://www.wangbase.com/blogimg/asset/201908/bg2019081405.jpg)

埃隆·马斯克（Elon Musk）的最新项目，是一家脑机接口公司，将大脑连入计算机。

他说，他很担心人工智能对人类的威胁，因此想到将人工智能接入人类体内，可能会缓解机器对人类的威胁。

他的新公司叫做 Neuralink（意为神经连接），必须通过手术，在人类体内植入硬件。这简直是将科幻小说的情节转变为一家公司，这到底是科学，还是更接近科幻小说？

![](https://www.wangbase.com/blogimg/asset/201908/bg2019081406.jpg)

Neuralink 的计划是，通过一个小于8毫米的洞进入大脑，将 4mm x 4mm 的芯片及其相关的线路植入大脑。该芯片将从位于耳后的无线硬件获得电源，并与之通信，就像当前的人工耳蜗植入一样。

在大脑内部，芯片连接到一些小线路，这些小线路连到放在各个区域的电极，监听神经元的电脉冲。芯片接收到神经活动的原始数据后，就会进行处理。同样的，电脉冲也可以通过这些电极发送到神经元，刺激大脑活动。

![](https://www.wangbase.com/blogimg/asset/201908/bg2019081407.jpg)

这个项目看上去更像科幻：电子植入物通过门诊手术放入大脑，通过无线方式与耳朵后面的硬件进行通信，全部由手机控制。但是，整个过程没有任何一个部分超出现有技术，所有挑战应该都可以克服。

除此之外，它的任何东西都显然属于科幻小说的范畴：将人类与计算机融合，抵抗人工智能。

这并不是说这些努力将是一种浪费，即使这些愿景都没有成功，依然有可能获得一些发现，解决一些不同的问题。正是因为他们选择解决一系列令人难以置信的挑战，无论他们最终解决什么问题，都可能是非常重要的。
