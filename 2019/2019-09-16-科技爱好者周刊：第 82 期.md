# 科技爱好者周刊：第 82 期

一个美国技术经理写了一篇[文章](https://erikbern.com/2019/09/12/misc-unsolicited-career-advice.html)，谈年轻人如何选职业。我觉得写得很启发思考。

首要的，他认为，要选择发展最快的行业。

> 停滞发展的公司，完全是零和游戏。如果你想晋升，必须等别人把位置空出来。你得到的，一定是其他人失去的。相比之下，快速发展的公司具有庞氏骗局的特性，源源不断的新人加入，最终每个人都会得到晋升！最重要的是，业务的发展比劳动力供给快，所以你会感到业务在“拉动“。

作者举了自己的例子。他并没有正式的机器学习背景，但是25岁就加入了一家机器学习公司，很快就组建了团队，成为技术管理者。因为周围找不到更好的人，所以环境会把你“逼”到这个位置上。

其次，就业的目的是为自己积累资本。有两种资本：人力资本（能力）和金融资本（钱）。年轻人应该关注快速形成自己的人力资本，而不是金融资本。因为长期来看，人力资本会比金融资本在你的一生中带来更大的红利。

最快速形成人力资本的方法，就是去聪明人多的地方，从比你更聪明的人身上学习。跟非常聪明的人一起工作，是最好的学习方法。如果你接触不到聪明人，即使跟他们在同一家公司里面，那也没用。

最后，如果这个行业里面，聪明人还不多，那是最理想的。一群聪明人在一个聪明人不多的行业中，可以创造奇迹。

## 文摘

1、[锂元素对人体的影响](https://literaryreview.co.uk/whats-in-the-7-up)

澳大利亚医生约翰·卡德（John Cade）通过发现锂改变了医学的发展方向。但大多数人从未听说过他。锂是心理健康方面的青霉素。它是人类发现的第一种用于治疗精神疾病的有效药物。

1949年，37岁的卡德发表了一篇论文，他发现锂元素可以使躁狂兴奋症患者冷静下来，症状平息。因此他认为，锂元素可以用来治疗躁狂抑郁症（现在通常称为躁郁症）和癫痫。

卡德来自一个医生世家，他的父亲担任过精神病院的医疗总监，所以他从小就了解精神病。他坚信，所有严重的精神疾病均归因于大脑中的生化异常，而不是心理因素导致的，精神分析法是无用的。

![](https://www.wangbase.com/blogimg/asset/201910/bg2019100501.jpg)

他做了一个实验，将躁狂症患者的尿液注射到豚鼠，结果豚鼠出现惊厥和死亡。他认为，躁狂症患者尿液中尿酸水平较高，导致豚鼠出现尿素中毒。根据文献，锂元素可以溶解含有尿酸的肾结石，他就向豚鼠注射了锂元素饱和的尿液，豚鼠没有惊厥或死亡。受到这一发现的鼓舞，他随后开始自己服用锂盐以测试其安全性，然后再对躁狂症患者进行观察性试验。

卡德的论文发表后，一些有好奇心的医生注意到了这篇文章。1950年代和1960年代，一位名叫 Mogens Schou 的精神病学家进行了大量的临床试验，结果表明锂元素不仅可以缓解急性躁狂症，还可以减少躁郁症的复发频率。他找了二十四位艺术家（包括作家、作曲家、画家），发现在具有严重情感障碍的艺术家身上。

越来越多的证据表明，锂是可以挽救生命的神奇药物，但它仍然是一种实验药物，许多知名的精神病医生都不愿使用它。

由于锂元素只需简单添加，不必开发出专门的药物，对于制药公司来说无法申请专利，也无利可图，且会打击市场上已有的药物，所以产业界不愿意推广它。布朗大学精神病学家沃尔特·布朗（Walter Brown）目睹了锂的积极作用后，伤心地写道，1960年代美国医药管理局 FDA 的拖延行为，使得成千上万的美国公民失去了一种能够防止破坏性的情绪波动，并降低自杀风险的治疗方法。

最终，锂元素终于投放了市场。布朗医生估计，20年中它为美国经济节省了约1,450亿美元。

早在1929年，一种含有镇静剂的柠檬石灰苏打水投放市场，它很快被重命名为7 Up（中文名为七喜），并成为一种流行的饮料。后来发现，它含有柠檬酸锂。

![](https://www.wangbase.com/blogimg/asset/201910/bg2019100502.jpg)

1948年，7 Up 的生产商从饮料中去除了锂。随着人们发现，世界上锂使用量较高的地区自杀率较低，7 Up 又重新加入了锂。

2、[伊万·奇索夫](https://en.wikipedia.org/wiki/Ivan_Chisov)

伊万·奇索夫（Ivan Chisov，1916–1986）是一个二战时期的苏联飞行员，从7000米高空自由落体下坠，却幸存下来。

![](https://www.wangbase.com/blogimg/asset/201910/bg2019100508.jpg)

1942年1月，一架德国空军的战斗机击中了奇索夫的轰炸机，迫使他在7000米的高空跳伞。

由于战斗机仍在他的上方，奇索夫一开始故意不打开降落伞，防止成为战斗机的目标。等到战斗机飞远了，他想打开降落伞的时候，已经来不及了，由于急速下坠，他陷入昏迷，失去了知觉，无法拉开降落伞的拉索。

奇索夫以大约190至240公里/小时的速度落地，撞到了一个积雪的山坡，然后一路向下滑行，滚到山沟底部。骑兵赶到现场，惊讶地发现他还活着，身上带着没有打开的降落伞。过了不久，奇索夫醒来，恢复了意识。

他受了重伤，包括脊柱受伤和骨盆骨折，紧急进行了手术。整整一个月，医生都认为他的病情很危险。但是，三个月之后，他恢复了健康，要求继续执行飞行任务。

战争结束以后，他进入了军事院校读书，后来成为苏军的人民代表。他于1986年去世。
