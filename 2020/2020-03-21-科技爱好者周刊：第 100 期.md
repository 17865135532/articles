2、[液态金属](https://www.pnas.org/content/117/10/5088)

![](https://www.wangbase.com/blogimg/asset/202003/bg2020032206.jpg)

1992年电影《终结者2》中，一个机器人杀手变形为液态金属，令观众印象深刻。科学家也对室温下能够导热和导电的液体，抱有浓厚的兴趣。汞虽然是液态的，但是对人体有毒。目前对液态金属的研究都集中在镓上，镓是无毒的，温度达到30°C就会熔化，更难得的是，镓合金的熔化温度更低，而且没有蒸气不会被吸入。

最近，中国研究团队已经证实，通过磁场可以控制镓-铟-锡合金，达到电影《终结者2》里面的液态金属受控聚合的效果。当一滴镓铟合金放在强碱溶液中时，施加电压会导致在该液滴从球形变成雪花状的分形图案（下图）。科学家认为这种行为有突破性的应用前景。

![](https://www.wangbase.com/blogimg/asset/202003/bg2020032207.jpg)

3、[iPad Pro 的新 CPU](https://mondaynote.com/arm-ed-mac-we-have-an-answer-545a20419a46)

![](https://www.wangbase.com/blogimg/asset/202003/bg2020032307.jpg)

苹果公司上周发布的 iPad Pro 支持触摸板和鼠标。国外评论认为，这实际上是苹果在尝试 ARM CPU 的笔记本电脑。如果成功，下一步就要将 MacOS 从 x86 CPU 移植到 ARM CPU。

苹果自己可以生产 ARM CPU，而 x86 CPU 都需要从 Intel 购买。更换 CPU 的主要好处是，ARM CPU 比较节能，电池可以使用更长时间。而且更换后，手机和笔记本的操作系统就可以统一了，理论上手机 App 就可以直接在笔记本运行。目前，主要指标是看 Xcode 何时能移植到 iPad Pro，现在还不行。

## 文摘

1、[为什么没有冠状病毒疫苗？](https://threadreaderapp.com/thread/1240498037958545410.html)

Covid-19 出现之前，常见的冠状病毒有4种，对于大多数人来说，仅仅引起轻度感冒。这些病毒会激发适度的免疫反应，足以克服感染，使你受到大约一年或更长时间的保护，但是体内的抗体最终会消失。 

也就是说，人类可以被同一种病毒再次感染。那么为什么我们不开发针对这4种冠状病毒的疫苗呢？

原因一。大多数感冒是由其他病毒引起的，比如鼻病毒、RSV、副流感病毒、其他未知的病毒等等，这4种冠状病毒只会导致大约20％的感冒。每一种病毒都需要不同的疫苗，所以实际上，冠状病毒疫苗需要4种。但是，即使你全部使用，也只能帮助您抵御 20％的感冒。

原因二。证明疫苗有效的临床试验，非常困难而且昂贵。即使疫苗将冠状病毒导致的感冒风险降低了75％，也仅将整体的感冒风险降低了15％。通过一大型试验，证明15％的减少是可能的，但会非常贵。而且，一个成年人每年得2次～4次感冒，所以减少15％甚至不等于每年少感冒1次，这听起来并不吸引人。
 
原因三。人们想要的不是普通的冠状病毒感冒疫苗，二是一种普通的感冒疫苗，可以抵御大多数感冒病毒，但是科学家至今也不知道30％的感冒是什么原因导致的。这意味着，疫苗必须具有多种成分（抗原）才能有效，所有的成员都需要单独的开发和研究，所以感冒疫苗并不是一种单一的产品，而是一种复合产品。 

原因四。大多数人对感冒并不重视，因为绝大部分感冒不会出现严重后果，所以他们不愿意出钱注射疫苗。感冒疫苗的市场前景并不好，葛兰素史克曾经开发过一种感冒疫苗，几年后就退出了市场。投资人不愿意支持商业失败的药物，所以一直以来感冒疫苗得不到足够的资金支持。
 
但是，Covid-19 疫苗不一样，它仅针对一种病毒，而不是十几种不同的病毒，所以试验会非常简单。并且，它的传染力强，对于老年人后果严重，保险公司和政府会愿意为它付款。
